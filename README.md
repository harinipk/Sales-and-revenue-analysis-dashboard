Sales & Revenue Analysis Dashboard

Project Overview

This project analyzes Blinkit sales data using Microsoft Excel.

Objectives

- Analyze sales and revenue trends
- Identify top-performing products
- Compare category-wise sales
- Analyze customer segments
- Create an interactive dashboard using Pivot Tables, Charts, and Slicers

Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards

Dashboard Features

- Total Revenue KPI
- Total Orders KPI
- Total Quantity Sold KPI
- Total Customers KPI
- Monthly Revenue Trend
- Top Products Analysis
- Revenue by Category
- Revenue by Customer Segment
